🎯 Objectives

    🚀 Quick Start

    Follow these steps in order. Requires a Linux host with Docker installed.

    01
    Check prerequisites

    Make sure Docker and Git are installed on your Linux machine before starting.

    bash
    docker --version && git --version

    ✦ Don't have Docker? Install Docker →

    02
    Clone & deploy the machine

    One command clones the repo and spins up the Docker containers in the background.

    bash
    03
    Verify the machine is running

    Confirm the containers started successfully. You should see them listed as Up.

    bash
    docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"
    04
    🎯 Start attacking

    The machine is live on localhost. Start with an nmap scan to map the attack surface.

    bash
    nmap -sV -sC -p- 127.0.0.1

    ✦ When done: docker compose down to clean up.
