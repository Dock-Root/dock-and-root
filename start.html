Requirements

What you need

🐳
Docker

Docker Engine 20.10+ with Docker Compose plugin. Runs on any modern Linux distro.

Install Docker →
🔧
Git

Git to clone the machine repositories. Available on every major OS out of the box.

$ apt install git
🧰
Your hacking toolkit

nmap, Burp Suite, netcat, sqlmap, gobuster — whatever you already use. No extras required.

Platform

Built different

🐳
Container-first, always

Every machine runs in Docker. No VM bloat, no hypervisor configs. Isolated environments that start and stop in under 10 seconds.

$ docker compose up -d
✔  Container db      Started   0.4s
✔  Container web-app  Started   0.6s
$ nmap -sV 127.0.0.1
80/tcp open http Apache 2.4.54
🔓
Always free

Open source under MIT. No paywalls, no premium tiers. Ever.

📡
Air-gap capable

Pull images once, hack offline forever. No telemetry, no cloud dependency.

🤝
Community-made

Machines contributed by the security community. Anyone can submit a challenge or write a walkthrough.

⚡
Real CVEs & techniques

Labs are modelled on real-world vulnerabilities and attack chains — not toy examples.

⚓
Open source

The whole platform — machines, website, tooling — is on GitHub. Fork it, improve it, contribute back.
